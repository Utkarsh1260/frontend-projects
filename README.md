# 📁 Project Overview

| Folder | Description |
|--------|-------------|
| `art-work` | Creative artwork-inspired landing page focusing on layout, typography, and modern UI design. |
| `astro-winter-store` | Winter clothing/e-commerce landing page with product showcase and responsive design. |
| `butusic-exhibition` | Exhibition or event website designed to present artworks, events, and information in a clean layout. |
| `Cloth-website` | Clothing/fashion website featuring responsive sections, product cards, and modern styling. |
| `digital-hunters` | Digital agency-style landing page showcasing services, animations, and responsive components. |
| `fintrack_pro_task_8` | Finance tracking dashboard built as a frontend task with a clean user interface. |
| `insect-details` | Informational website displaying insect-related content using cards and structured layouts. |
| `insights-and-blogs` | Blog website with article sections, cards, categories, and responsive design. |
| `john-portfolio` | Personal portfolio website showcasing projects, skills, and contact information. |
| `lombok` | Travel and lifestyle-inspired landing page recreated for frontend practice. |
| `microsoft-landing-page` | Frontend clone inspired by Microsoft's official landing page. |
| `productivity_dashboard_task_9` | Productivity dashboard UI displaying widgets, statistics, and task management components. |
| `squid-game` | Landing page inspired by the Squid Game theme with custom styling and visual effects. |
| `task_manager_application_task_7` | Frontend task manager application with task management interface and interactive features. |
| `the-face.com` | Creative website inspired by modern portfolio and agency designs. |
| `twoleaf.com` | Business/brand landing page featuring responsive sections and modern UI components. |
